GIF Dance Party Extension

written 2015-12-09

Jimmy and I have finished the GIF Dance Party Chrome extension and it is now available in the chrome webstore. Enjoy!

Chrome Webstore

The Extension

The GIF Dance Party Extension is based on Gif Dance Party, a website where you can position dancing GIFs along with music. The extension fixes a lot of the technical glitches with the site itself, and allows you to create a "party" on any webpage by clicking the extension icon.

Development

Github

The extension makes heavy use of JQuery and JQuery UI to make the editable GIF tiles on screen, as well as create the menus. We're using the chrome browser process messaging system to trigger Javascript audio streaming in a background process. The background process is also what registers the click on the icon to start the party. We discovered a limitation of Javascript audio however: currently looping audio is not at all seamless which does hurt the experience.

Another neat feature we added was the ability to add custom GIFs and audio files beyond what we put in the menus. Each addition is stored in the Chrome data storage and persists between uses.

Future

I'm currently looking in to creating a packaged version of the extension that can be embedded in to webpages by site owners. Insert "start the party" button here.
